Translation (Chinese)

Translation is optional and disabled by default. When enabled, WordMark will only make a network request after you explicitly click Translate in the lookup overlay. The request sends only the looked-up word and (if present) the definition text.

Volcengine configuration is required only when Volcengine is selected.
Zhipu configuration is required only when Zhipu is selected.
API key is stored locally and never displayed back in full.
